기존 에디터의 기능은 다음과 같아. 

1. 디비로부터 영역 데이터를 받아와서 어떤 영역 데이터가 있는지 파악
2. 유저는 기본씬에 특정 영역을 선택(예를 들어 1920x158)하고 해당 에디터 폼에 이미지, 동영상, 웹페이지를 등록
3. 씬은 최대 10개까지 영역을 구분할수 있으며 각 영역은 드래그해서 위치 조절 가능하고 해당 사이즈가 표시됨. 
4. 각 분할된 영역에는 각기 다른 컨텐츠가 들어감. 
5. 각 분할된 영역은 드래그앤 드롭으로 순서를 변경할수 있음.  
5. 씬추가가 가능하고 추가된 씬은 활성화 시간을 주어서 번갈아서 실행이 되도록 함. 
6. 각 씬은 순서 변경이 가능함. 
7. 작업된 씬은 html로 저장되는데 리소스는 모두 base64로 내부에 심음 


이제 우리가 해야할 작업은 다음과 같아. 

1. 현재는 씬 에디터의 영역에 각 컨텐츠가 하나씩만 들어 갈수 있는데 이를 레이어 형태로 여러개를 입력 할수 있도록 기능 추가 
2. 예를 들자면 영역에 이미지를 처음 삽입하면 그 이미지는 배경화면이 되고 그 위에 다른 이미지나 텍스트를 덧 씌울수 있도록 함. 
3. 배경화면 위에 올라가는 이미지는 사이즈 조절, 위치 조절이 가능해야 함. 
4. 텍스트도 텍스트 영역을 지정 할수 있어야 하고 배경색, 폰트, 텍스트 크기, 텍스트 색깔, 텍스트 위치등을 변경할수 있어야 함. 

순차적으로 작업을 진행해 보자.
